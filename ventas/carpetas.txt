1.- bd: carpeta que contiene los snipets o querys de la bd listoa para ejecutar.
2.- clases: contiene todas las clases de cada modulo, con sus respectivos metodos de crud.
3.- css: contiene los scripts css que serviran para dar estilo personalizado en caso de ser util.
4.- img: contiene todas las imagenes que necesitamos agregar a nuestras  vistas.
5.- archivos: contiene todas las imagenes de los productos agregados,
6.- js: contiene todos los scripts js para cada modulo para ejecutar las funciones de crud.
7.- librerias: contendra librerias para la creacion de pdf, jquery, alertify, boostrap y select 2.
8.- procesos: contiene archivos intermedio entre las clases y las vistas, estos mandaron informacion entre una y otra.
9.- views: contendra todas las vistas de cada uno de los modulos.
